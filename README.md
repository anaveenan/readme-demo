# Numberstring

The `numberstring` package can transalate integers into string in python  

## Installation 

You can install via `pip` 

```
pip install numberstring
```
## Usage

```python
import numberstring

numberstring.translate(1) # "one
numberstring.translate(101) # "one-hundred"
```
